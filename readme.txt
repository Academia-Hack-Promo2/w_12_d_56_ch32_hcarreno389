
Registro de Personas con AJAX

Construya un formulario para el registro de Personas, el cual sea validado por JQuery, y se envíe la data mediante Ajax al Servicio destinado a almacenar los registros, el servicio responderá con TRUE si la operación fue exitosa o FALSE si la operación fue errada. Debe mostrar en un Dialogo Modal el resultado de la operación, de ser exitosa la operación se debe tener un vinculo para ir al listado de las personas registradas.

El listado de Personas registradas debe consultarse mediante un servicio que devuelve el listado en forma de JSON Este listado debe ser mostrado en una tabla HTML en la página correspondiente.
